Capabilities

Not just another chatbot quiz.

Built on Hermes Agent's unique architecture — the learning loop, persistent memory, and multi-platform delivery make this fundamentally different from static question banks.

🧠

Learning Loop

Your interview profile persists across sessions. Questions adapt to target your weak areas. Difficulty auto-scales as you improve. The skill literally gets better at coaching you over time.

📊

Scored Feedback

Every answer evaluated across 5 dimensions with company-specific rubrics. Amazon Leadership Principles, Google Googleyness, Stripe Rigor — each adds relevant scoring criteria.

🔍

Company Research

Set a target company and get real-time interview intel via web search — common questions, process stages, culture signals, red flags. Research caches locally for instant access.

⏰

Daily Practice

Cron-scheduled daily challenges via Telegram, Discord, Slack, or WhatsApp. One question each morning, weighted toward your weaknesses. Reply and get instant feedback.

📈

Progress Tracking

Visual progress reports with score trends, category breakdowns, and actionable recommendations. Weekly digests. Interview readiness assessment against company-specific bars.

🌍

Works Everywhere

Practice from CLI, Telegram, Discord, Slack, or WhatsApp — your progress follows you. Start a session on your laptop, continue on your phone during commute.

Architecture

How it uses Hermes Agent

Every major Hermes capability is doing real work — this isn't a wrapper around a prompt.

Hermes Feature How Interview Coach Uses It
Learning Loop Profile updates after every session; question selection adapts to target weak areas; difficulty scales automatically
Persistent Memory Interview profile, session logs, and company research survive restarts, model switches, and platform changes
Tool Use Web search for company research; file I/O for progress tracking; code execution for technical evaluation
Cron Scheduling Daily practice questions delivered on any connected platform — weekday mornings, configurable
Multi-Platform Practice from Telegram, Discord, Slack, WhatsApp, or CLI — one profile across all
Skills System Self-contained skill with auto-improving question banks; compatible with agentskills.io
User Modeling Tracks strengths, weaknesses, target companies, preferred difficulty, and learning trajectory
Subagent Delegation Parallel company research while conducting interviews; batch evaluation for session summaries
